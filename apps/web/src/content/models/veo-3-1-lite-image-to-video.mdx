## Animating one starting image

Veo 3.1 Lite supports image-to-video generation from an initial still frame and a text prompt. The image defines the opening composition, subject detail, and visual style. The prompt should describe the movement that follows, including the subject’s action, camera motion, environmental changes, timing, and sound. Repeating every visible feature of the image is less useful than explaining how the scene should develop.

The output includes synchronized audio automatically. A creator can request dialogue, effects, music, or ambience alongside the visual motion, but all of it needs to fit a clip lasting four, six, or eight seconds. Results use either 16:9 landscape or 9:16 portrait framing and play at 24 frames per second.

## Controls specific to Lite

Veo 3.1 Lite supports 720p and 1080p. The higher resolution is limited to the eight-second duration; shorter clips use 720p. Google’s feature matrix does not list the multi-image reference control or video-extension input for Lite. That means this workflow should be understood as animation from a single opening image, not the broader style-reference or video-to-video system available to standard Veo 3.1.

The person-generation setting for image-conditioned Veo requests is documented as `allow_adult`. A negative prompt can identify unwanted elements. A seed may make repeated attempts somewhat more consistent, although it does not make generation deterministic.

## Preparing the frame and prompt

The source image should already have the intended landscape or portrait composition because it becomes the first frame. The prompt can then focus on velocity, direction, camera language, and sound rather than asking the model to rebuild the scene. Only one video is returned per request, and generation runs asynchronously. Veo 3.1 Lite remains in preview and generated videos include SynthID watermarking. Its limited reference surface and 1080p maximum distinguish it from the standard Veo 3.1 image workflow while keeping the core first-frame animation path intact.
