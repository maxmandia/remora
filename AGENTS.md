# Remora

Remora is aiming to become the Linear of generative media creation: a refined, fast, highly engineered workspace for creating image and video assets through multiple model providers.

The current codebase is intentionally a bare skeleton: TanStack Start for the web app, Fastify/tRPC for the API shell, a future worker boundary for orchestration, and shared Drizzle/env packages. Keep early changes lean, typed, and biased toward clean service boundaries without adding product behavior before it is needed.
